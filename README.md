# Portfolio artistique — Liz Barthel

## Contexte
Projet de refonte et de restructuration d’un portfolio artistique.
Objectif : valoriser le travail de l’artiste, améliorer la lisibilité,
faciliter les collaborations, les expositions et la vente sur demande.

## Rôle
- Conception de la structure du site
- Organisation des œuvres par séries
- Création d’une galerie responsive
- Mise en place d’un parcours narratif (parcours artistique)
- Design orienté visibilité et conversion élégante
- Intégration HTML / CSS / JavaScript sans CMS

## Approche
Le site a été pensé comme un espace de découverte :
une “ville” dans laquelle le visiteur circule librement entre les œuvres,
le parcours et l’univers de l’artiste.

Priorité donnée à :
- la clarté
- la respiration visuelle
- la mise en valeur des œuvres
- une vente non intrusive (contact direct)

## Technologies
- HTML5 sémantique
- CSS3 (Grid, Flexbox)
- JavaScript vanilla
- Git / GitHub

## Résultat
- Site rapide, léger et maintenable
- Structure claire et évolutive
- Base solide pour SEO, blog et automatisation future
- Portfolio utilisable pour présentation professionnelle

## Statut
Projet en cours d’affinage avec validation artistique.
